# 回合制战斗游戏 (Turn-Based Battle Game)

一个基于浏览器的回合制战斗游戏，包含四个独特角色，每个角色都有独特的技能和魂印效果。

## 特色

- 🎮 四个可玩角色，各具特色的技能系统
- ✨ 魂印系统 - 每个角色独特的被动效果
- 💫 丰富的战斗机制：状态效果、能力强化、护盾系统
- 🎨 现代化的UI设计，动态效果反馈
- ⚔️ 多样化的敌人AI和技能

## 角色

1. **王·盖亚** - 战斗型角色，擅长反弹和防御
2. **不灭·艾恩斯** - 火焰系，拥有不死效果
3. **怒涛·沧岚** - 水系，护盾和控制专家
4. **混沌魔君索伦森** - 混沌系，能力同步和强化消除

## 如何游玩
